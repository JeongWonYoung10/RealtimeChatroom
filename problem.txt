Build a visual workflow simulator for a small job queue. Goal: Create a browser app that simulates jobs moving through a processing pipeline. Requirements: User can configure arrival rate, worker count, processing time range, failure rate, and retry limit. Jobs move through states: queued, processing, succeeded, failed, retrying, dead-letter. The simulation runs step by step and in autoplay mode. Show live metrics: queue length, throughput, average wait time, failure count, retry count. Render a visual pipeline with moving jobs or state columns. Allow pause, reset, and random seed control. Include an event log. Keep it static HTML/CSS/JS with no backend. Persist the last configuration in localStorage.
